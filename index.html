Intro to HTML

What is HTML?

HTML stands for HyperText MarkUp Language and is the standard markup language for documents designed to be displayed in a web browswer. It can be assisted by technologies such as Cascading Styles Sheets and scripting langugages such as Javascript.

All html documents must be first classfied with a <!DOCTYPE html>. Followed then by a <head> which would support all of the Metadata not shown on the webpage and then a <body> which will hold everything that displays on the homepage.

Tag Name Description
<head> Head The head element is a container for metadata (data about data) and is placed between the tag and the before the tag. Metadata is not displayed on the page. Metadata typically define the document title, character set, styles, scripts, and other meta information.
<body> Body The body element contains all the contents of an HTML document, such as headings, parapgraphs, images, hyperlinks, tables, lists, etc. Note there can only be one body element
<div> Div tag The div tag defines a division or a section in an HTML document. The tag is used as a container for hTML elements which then can be styled in css through class or ids.
<!--...--> Comment Allows you to write a comment in the html script without in appearing on the webpage
<h1> to <h6> Headings The headings in html are six levels of headings. A heading element implies all the font changes, paragraphs breaks before and after and any white space necessary to render the heading. The H1 heading is the largest and H6 is the smallest.
<p> Paragraph The paragraph is as mentioned, a paragraph in html. It starts on a new line and browsers automatically add some white space (a margin) before and after a paragraph
<a> Anchor The anchor element with href,creates a hyperlink to web pages, files, email addresses, locations in the same page defined with a class, or anything else a url can address.
<br> Break Inserts a single line break on page.
<em> Emphasize Used to emphasize text
<img> Image Image tag is used to embed an image in an HTML page. Images are not techinically inserted into a web page; images are linked to web pages. Usually to link an image the code will look like: <img src="(your image url here)" alt="photo explaination for accessibly">
<figure> Figure The figure tag specifies self-contained content, like illustrations, diagrams, photos, codes listings, etc. While the content of the <figure> element is related to the main flow, its position is independent of the main flow, and if removed it should not affect the flow of the document.
<figcaption> Figure caption Figure caption represents a caption or legend decribing the rest of the contents of its parent <figure> element.

For video code usually you would add the following:

"<video width="200 height="150 controls><source src=vid.mp4 type="videomp4">No video support</video>

and for audio it would look like the following

"<audio controls><source src="sound.ogg" type="audio/orr"> No audio support</audio>

Intro to CSS

What is CSS?

CSS is a style sheet langugage used for decribing the presentation of a document writtern in a markup language such as HTML. CSS is usually connected to HTML by adding the following <link rel="stylesheet" href="./name of folder" type="text/css">into the header

A css rule consist of a selector and a declaration block which could consist of property: value. An example of a CSS syntax is: h1{color:blue}.

Height & Width

The height and width of properties are used to set the height and width of an element. These do not include margin, padding and borders, they set the height and width of the content inside.

Positioning

Position property specifies the type of positioning method used for element (static, relative, fixed, absolute or sticky). Elements are then positioned using top, bottom, left and right properties- however these properties will not work unless the position property is set first.

display:inline-blockInline-BlockDisplays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values.
Position Name Description
position:static Static For the static position elements are not effected by the top, bottom, left, and right properties. A element with position:static; is not positioned in any special way; it is always positioned according to the normal flow of the page.
position:relative Relative An element with position:realtive is positioned relative to its normal position. Setting the top, right, bottom and left properties of a relatively-positioned element will cause it to be adjusted away from its normal position. Other content will not be adjusted to fit into any gap left by the element.
position:fixed Fixed An element with position:fixed; is a positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled. The top, right, bottom and left properties are used to position the element. A fixed element does not leave a gap in the page where it would normally have been located.
position:absolute Absolute An element with position:absolute; is positioned to the nearest positioned ancestor (instead of positioned relative to the viewport, like position:fixed). However, if an absolute positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling. NoteAbsolute positioned elements are removed from the normal flow, and can overlap elements.
position:sticky Sticky An element with position:sticky; is positioned based on the user's scroll position. A sticky element toggles between relative and fixed, depending on the scroll position. It is positioned relative until a given offset position is met in the viewport - then it sticks in place (like position:fixed).
z-index: Z Index When elements are positioned, they can overlap other elements. The z-index property specifies the stack order of an element (which element should be placed in front of, or behind, the others). A element can have a positive or negative stack order. Example (z-index: 6 will be in front, followed by z-index: 4, z-index:2, z-index:-1, z-index:-4, z-index:-6 behind the element)
overflow: Overflow The overflow property controls what happens to the content that is too big to fit into an area. Overflow specifies whether to clip the content or to add scrollbars when the content of an element is too big to fit in the specified area. The overflow property has the follow values: visible: default, overflow is not clipped. hidden: overflow is clipped, and the rest of the content will be invisable. Scroll: The overflow is clipped and a scrollbar is added to see the rest of the content. auto: similar to scroll but adds scrollbars only when necssary.
float: Float The float property is used for positioning and formatting content eg let an image float left to the text in a container.
display:inline Inline Displays an element as an inline element (like<span>). Any height and width properties will have no effect.
